# Shared Prefabs

Place shared Bannerlord prefabs for Warband-to-Bannerlord mappings in this folder.

The converter does not automatically load files from this folder. It is a repo location for prefabs
that users want to share alongside `asset_mappings.xml`, so other converters can use the same
Bannerlord prefab IDs when mapping Warband scene props.

Suggested use:

1. Add reusable Bannerlord prefab files here.
2. Reference their prefab IDs in the converter GUI.
3. Share mapping updates together with the prefab files they require.
